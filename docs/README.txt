COMPATIBILITE
=================
Ce site utilise de nombreuses fonctionnalités CSS qui peuvent ne pas être supportées par tous les navigateurs (notamment backdrop-filter).
Ce site a été optimisé pour les navigateurs utilisant le projet Chromium, comme Microsoft Edge (v79 et plus) et Google Chrome.
Ce site n'est pas adapté pour les appareils mobiles.
